# SEO launch checklist

Use this after the production domain and SSL are working.

1. Create a Google Search Console property for `https://apexgloballogistics.net`.
2. Choose the HTML meta tag verification method and copy only the verification token.
3. Add the token to `GOOGLE_SITE_VERIFICATION` in `.env.production`.
4. Recreate the app container so the metadata is rebuilt.
5. Submit `https://apexgloballogistics.net/sitemap.xml` in Google Search Console.
6. Inspect the home page URL and request indexing.
7. Create or claim the business profile and keep the name, website, support email, and public contact details consistent.
8. Collect only genuine reviews from real customers using neutral invitations.

Never buy fake reviews. Use Trustpilot or Google review invitations only for customers who actually used the service.
